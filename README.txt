################## Installation ###################
Pour installer et utiliser le module, veuillez effectuer les actions suivantes :
  1 - Telecharger le module (fff_module_chrome.zip) a l'adresse suivante 'https://dl.dropboxusercontent.com/u/25112915/fff_module_chrome.zip'
  2 - Mettre le ficher dans un endroit que vous connaissez (ex: Mes Documents)
  3 - Dezipper le fichier
  4 - Se rendre sur le naviguateur 'Google Chrome', et saisir 'chrome://extensions/' dans la barre d'adresse
  afin d'afficher les modules prealablement installes sur votre ordinateur
  5 - Cliquer sur le bouton 'Charger une extension deballee...' puis selectionez le repertoire 'fff_module_chrome' 
  (repertoire contenu dans l'archive dezipee)
  6 - Clique droit sur le logo en haut a droite de votre naviguateur (logo FFF) puis cliquez sur 'Options'
  7 - Saisissez votre numero de licence d'arbitre et votre mot de passe et cliquez sur "Enregistrer mes informations"
  8 - Et voila !! Votre module est operationnel. Testez le en cliquant sur le logo FFF en haut a droite de votre naviguateur,
  un onglet s'ouvrira et affichera directement vos designations.
